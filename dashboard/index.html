Frequently Asked Questions

What is an MCP server? expand_more
MCP (Model Context Protocol) servers are AI tool integrations that allow language models to interact with external services, APIs, and data sources. MCP Factory automates the creation of these servers from requirement specifications.
How long does a build typically take? expand_more
Most builds complete within 15-45 seconds, depending on the complexity of your PRD specification and the number of tools being generated. Complex MCPs with many integrations may take longer.
Can I use my own PRD template? expand_more
Yes! You can upload your own PRD in YAML or Markdown format using the "Upload PRD" option. Make sure to follow our template structure for best results. Check the Documentation section for template guidelines.
Where are my MCPs stored? expand_more
Generated MCPs are stored in the output directory configured in your settings (default: /home/ubuntu/projects/mcp/). They can also be automatically pushed to your configured GitHub repository.
